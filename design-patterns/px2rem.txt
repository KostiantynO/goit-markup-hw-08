Есть 2 варианта, которые я могу придумать сразу же.

1: функция CSS calc ()
$select-height: calc(.75rem + 2px);


2: преобразовать пиксели в rem
@function rem($px, $base: 16) {
  @return $px / $base;
};
$select-height: #{.75 + rem(2)}rem;

Если 16px - это основа для моих единиц rem,
которые я могу изменить, передав второй аргумент в мой вызов rem. Удаление единиц заставляет математику работать.

В качестве альтернативы вы можете использовать аналогичную функцию для преобразования вашего rem в px, переключив деление на умножение.
Затем вы можете преобразовать все обратно в rem.

Но это самый простой из известных мне методов, если вы занимаетесь математикой rem-to-px.



Если бы у вас была другая переменная для вашего базового размера в пикселях, вы могли бы запустить функцию strip-units следующим образом:

@function strip-units($value) {
  @return $value / ($value * 0 + 1);
};
